Architecture Overview
The CNN-BiLSTM model (Algorithm 1, Section III.C) integrates:
• CNN Stage: 3 Conv1D layers (64→128→256 filters, kernel=3) + ELU + AvgPool
• BiLSTM Stage: 2 bidirectional LSTM layers (128 units each)
• Dual Output: Classification (Softmax) + Morphology Prediction (Linear, 47 PCA coefficients)
• Total Parameters: 952,507 (Table IV)

Model Architecture Details
Algorithm 1: CNN-BiLSTM Inference Pipeline
Implemented in model.py — CNNBiLSTMSpikeSorter class:
# Stage 1: Spatial Feature Extraction (CNN Layers 1-9)
h1 = Conv1D(x, kernel=3, filters=64) # Layer 1
h2 = ELU(h1, alpha=1.0) # Layer 2
h3 = AvgPool(h2, pool=2) # Layer 3
h4 = Conv1D(h3, kernel=3, filters=128) # Layer 4
h5 = ELU(h4, alpha=1.0) # Layer 5
h6 = AvgPool(h5, pool=2) # Layer 6
h7 = Conv1D(h6, kernel=3, filters=256) # Layer 7
h8 = ELU(h7, alpha=1.0) # Layer 8
h9 = GlobalAvgPool(h8) → 256-dim vector # Layer 9
# Stage 2: Temporal Modeling (BiLSTM Layers 10-13)
h10 = Reshape(h9, to=[1×256]) # Layer 10
Generated by Kimi.ai
[h→, h←] = BiLSTM(h10, units=128×2) # Layer 11
h12 = Dropout([h→, h←], rate=0.3) # Layer 12
[h→, h←]₂ = BiLSTM(h12, units=128×2) # Layer 13
h13 = Dropout([h→, h←]₂, rate=0.2) # Layer 14-prep
# Stage 3: Dual-Output Generation (Layers 14-16)
z = Dense(h13, units=128, ReLU) # Layer 14
z_drop = Dropout(z, rate=0.2) # Layer 15
# Output 16a: Classification
ŷ = Softmax(Dense(z_drop, units=K)) # K neuron classes
# Output 16b: Morphology Prediction
ŷ_pred = Linear(Dense(z_drop, units=47)) # 47 PCA coefficients
Key Design Choices
• Sequence length: 16 consecutive spikes (Section III.C.2)
• Zero-padding: Left for first 15 spikes, right for final spikes
• Prediction scope: Single-step forecast of next detected spike’s morphology
• Prediction invalidation: If ISI > 500ms (Section III.C.3)
• Value clipping: [-5, +5] standard deviations during training

Known Limitations
From Section VI.B:
1. Computational Cost: 2.3×10￿ FLOPs — requires GPU acceleration
2. Predictive Limitation: Predicts morphology (PCA coefficients) only, not ISI or spike timing
3. Cross-Dataset Generalization: 12.2% drop (motor cortex → hippocampus), 14.7% drop (rodent
→ primate)
4. Supervision Dependency: 96.4% accuracy requires full labels; 10% labels yield 88.9%
5. Single-Electrode Processing: 12% over-splitting for multi-electrode recordings
6. Reproducibility: Proprietary MEA dataset restrictions (see Step 2 in remediation plan)
