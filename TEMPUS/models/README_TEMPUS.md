# TEMPUS: TEmporal Morphology Prediction with Uncertainty for Spikes

## Novel Algorithm for Real-Time Spike Sorting

TEMPUS is a novel algorithm for real-time spike sorting with 
uncertainty-aware morphology prediction. It introduces **four algorithmic 
innovations**:

1. **Adaptive Spike Attention (ASA)** - Firing-rate-adaptive attention mechanism
2. **Spike-Specific Evidential Regression (SSER)** - Class-conditioned uncertainty
3. **Temporal Consistency Regularization (TCR)** - Semi-supervised learning
4. **Physiologically-Constrained Loss (PCL)** - Neurophysiological constraints
